Interview questions

1. What are pipes in Angular?

  • Pipes are special functions that can be exectued from view/HTML,
    but the syntax is little different when comparing to normal function call
  • We have to use | symbol for consuming a pipe
  • Syntax: {{ "{{ accountBalance | currency: 'INR' }}" }}
  • We also have some built-in pipes. We can also create custom pipes in Angular.
  • We are also going to see some example to understand pipes.

2. Name some built-in pipes?

  • lowecase
  • uppercase
  • json
  • date ("MM/dd/yy", "h:mm a", "long", "short", "shortDate" )
  • currency (:'EUR':'Euros ')

3. Why do we need pipes?

  • Pipes are re-usable functions
  • Pure pipes can improve the application performance

4. Pipe chaining

  • {{ "{{ name | uppercase | lowecase }}" }}

5. Pure vs Impure pipes?

{{ name | uppercase }}

{{ upperCase(name) }}

{{ name | upper }}
